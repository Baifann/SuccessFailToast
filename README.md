# SuccessFailToast

---

### 自定义的toast

#### 使用方法（成功状态）
```java
SuccessOrFailToast.getInstance(MainActivity.this).show("成功", SuccessOrFailToast.STATE_SUCCESS);
```

#### 失败状态
```java
SuccessOrFailToast.getInstance(MainActivity.this).show("失败", SuccessOrFailToast.STATE_FAIL);
```
